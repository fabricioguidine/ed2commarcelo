# Output Files

This directory contains output files generated by the experiments.

## Files Generated

- `saida.txt` - Sorting algorithm performance results (Part I)
- `saidaPart2.txt` - Most frequent authors results (Part II)
- `saidaInsercao.txt` - Tree insertion performance results (Part III)
- `saidaBusca.txt` - Tree search performance results (Part III)

## Results Organization

Results are also organized in subdirectories:
- `tests/sorting/` - Sorting algorithm detailed results
- `tests/hashtable/` - Hash table experiment detailed results
- `tests/trees/` - Tree structure experiment detailed results
